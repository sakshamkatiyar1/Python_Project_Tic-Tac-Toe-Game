# Tic-Tac-Toe Game (Python + Tkinter)

A simple Tic-Tac-Toe GUI game built using Python’s tkinter library. This project demonstrates basic GUI development, event handling, and game logic implementation.

# Features

 Interactive GUI using Tkinter
 Two-player game (X vs O)
 Automatic winner detection
 Winning combination highlighted in green
 Player turn indicator
 Simple and clean code structure

# Technologies Used

Python

Tkinter (GUI Library)

# Game Rules

The game is played between two players: X and O
Players take turns marking a cell
First player to align 3 marks (row, column, or diagonal) wins
The game ends when a player wins
